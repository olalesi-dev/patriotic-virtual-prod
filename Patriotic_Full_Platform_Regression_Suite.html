4. Prescriptions Flow

TC-012 Module: ePrescribe
DoseSpot e-Prescribing Workflow & Audit Trail
Preconditions: Provider is registered via SSO DoseSpot config. Patient record is open.
Steps:
  1. Click "New Prescription" from the clinical chart.
  2. Confirm DoseSpot interface authenticates and loads.
  3. Select Medication (e.g., Amoxicillin 500mg), dosage, and duration.
  4. Select a test Pharmacy and click "Send".
  5. Close DoseSpot frame and view local Medications list in EMR.
Expected Result: Prescription is authorized through the external network (Surescripts). EMR local UI reflects the new active medication with an audit log showing Date/Time.
